# simpleCounter
This is a simple 8bit counter with load/up/down functions.<br>

## How to start Simulation
Please use Icarus Verilog and GtkWave. To install them...<br>
```bash
sudo apt install iverilog gtkwave
```
To start simulation...<br>
```bash
cd SIM
./go_sim
```
To show the waveform...<br>
```bash
gtkwave tb.vcd tb.gtkw
```
